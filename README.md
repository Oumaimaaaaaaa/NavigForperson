# Navigate

## Project idea

A simple English-language demo that shows how to estimate the exposure of your personal information online through a small form.

## Current features

- Input form: full name, email, phone number
- Quick exposure score calculation
- List of exposure indicators
- Practical privacy tips

## Run locally

Open [index.html](index.html) directly in the browser or run a simple server:

```bash
python -m http.server 8000
```

ثم افتح: http://localhost:8000
